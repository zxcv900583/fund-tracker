| 基富通格式匯出 | 「CSV」對話框內可匯出與基富通完全同格式的 27 欄報表（含四種含息/含手續費損益與報酬率變化、台幣本金換算），檔名沿用「投資總覽_全部基金_匯出報表_YYYYMMDD.csv」，可再匯回（round-trip 驗證通過） |

## 1. 專案概述

### 1.1 背景

目前透過基富通證券網站管理基金投資組合，**必須登入才能查看淨值與損益**。本專案目標是打造一個**免登入、免安裝、手機可用**的個人基金追蹤工具，行為類似桌面版券商軟體（參考：Internet Quotes Assistant 截圖），但以 PWA 網頁形式呈現。

### 1.2 核心目標

| 目標 | 說明 |
|------|------|
| 免登入 | 不需任何帳號，開啟即用 |
| 免安裝 | 純網頁 PWA，手機可加入主畫面 |
| 本地持久化 | 投資組合資料存於瀏覽器 localStorage |
| 自動追蹤淨值 | 呼叫公開 API 取得最新每日淨值 |
| 損益計算 | 依買入記錄計算報酬率、損益金額 |
| 歷史折線圖 | 多種時間範圍切換的淨值走勢圖 |

### 1.3 參考 UI

參考截圖為 **基富通_翔.IQA - Internet Quotes Assistant**，其核心版面如下：

- 上方：投資組合**列表表格**（每行一支基金），顯示最新淨值、損益%、持有單位數、投資金額等
- 下方：點選任一基金後顯示該基金的**歷史淨值百分比折線圖**（以買入均價為 100% 基準）
- 底部狀態列：顯示總投資金額與總損益

## 2. 技術架構

### 2.1 形式

**純前端 PWA（Progressive Web App）**

- 單一 HTML 檔案（或 Vite/React 打包）
- 無後端伺服器
- 手機瀏覽器可「加入主畫面」離線使用（Service Worker）
- 淨值資料仍需網路才能更新

### 2.2 技術選型建議

| 層次 | 建議技術 | 備註 |
|------|----------|------|
| 框架 | React 18 + Vite | 或純 HTML/JS 亦可 |
| 圖表 | Chart.js 或 Recharts | 輕量、手機友善 |
| 樣式 | Tailwind CSS | 快速 RWD |
| 儲存 | localStorage | 無帳號版本 |
| API 呼叫 | fetch() | 瀏覽器原生 |
| PWA | vite-plugin-pwa | Service Worker 自動生成 |
| 搜尋 | TDCC OpenAPI | 見第 3 節 |

### 2.3 跨域問題（CORS）說明

> **重要：** `openapi.tdcc.com.tw` 的 CORS 政策未必允許前端直接呼叫。
> 若遇到 CORS 錯誤，需在中間架設一個**輕量 Proxy**（可用 Cloudflare Worker 或 Vercel Edge Function，免費方案足夠），轉發 TDCC API 請求。

## 3. 資料來源 & API 規格

### 3.1 主要資料源：臺灣集中保管結算所 OpenAPI

**Base URL：** `https://openapi.tdcc.com.tw`  
**文件：** `https://openapi.tdcc.com.tw/tdcc-opendata-api-docs`  
**授權：** 政府資料開放授權條款第 1 版（免費、免 API Key）

#### 3.1.1 國內基金（境內基金）淨值

> TDCC OpenAPI 目前公開的是**期信基金**（期貨信託基金）淨值（endpoint `/v1/opendata/5-4`）。
> 一般境內股票型基金（如 UNI023 統一奔騰）屬於**證券投資信託基金**，TDCC 本身不直接提供單檔查詢 endpoint，需使用以下替代方案。

#### 3.1.2 境外基金淨值

TDCC OpenAPI 提供境外基金淨值查詢，可直接呼叫：

```
GET https://openapi.tdcc.com.tw/v1/opendata/1-1
```

回傳欄位：`資料日期`、`基金代碼`、`基金名稱`、`計價幣別`、`淨值`、`淨值日`

#### 3.1.3 境內基金（一般投信基金）淨值 — 替代方案

由於 TDCC OpenAPI 對境內基金沒有開放單一代碼查詢，採用以下策略：

**方案 A（推薦）：政府資料開放平台**

```
GET https://data.gov.tw/api/v2/rest/datastore/11454?
    filters=基金代碼:UNI023
    &limit=1
    &sort=淨值日 desc
```

> 資料集 ID `11454` 為「國內基金淨值」，由金管會每日更新。

**方案 B：SITCA 投信投顧公會網站爬取**

- URL：`https://www.sitca.org.tw/ROC/Industry/IN2105.aspx`
- 方式：POST 表單，需 proxy 繞過 CORS
- 可取得歷史淨值資料（完整歷史）

**方案 C（備援）：Morningstar 非官方 API**

```
GET https://lt.morningstar.com/api/rest.svc/timeseries_price/
    9vehuxllxs?id=F000009XIM:NTW:TWF
    &currencyId=TWD&idtype=Morningstar&frequency=daily
    &startDate=2025-01-01&endDate=2026-06-10&outputType=JSON
```

> 基富通的基金頁面資料即來自 Morningstar，格式較穩定但非官方授權。

### 3.2 基金搜尋

**方案：呼叫 TDCC 基金清單，前端搜尋**

1. 啟動時一次性載入全部基金清單（TDCC `/v1/opendata/5-1` 或政府開放平台基金基本資料）
2. 前端 fuzzy search（依代碼或名稱）
3. 使用者選擇後記錄至 localStorage

## 4. 資料儲存設計

### 4.1 localStorage Schema

所有資料存於 `localStorage`，key 前綴統一使用 `fund_tracker_`。

#### 4.1.1 投資組合持有清單

**Key：** `fund_tracker_holdings`  
**型別：** JSON Array

```json
[
  {
    "id": "holding_001",
    "fundCode": "UNI023",
    "fundName": "統一奔騰基金",
    "currency": "TWD",
    "purchases": [
      {
        "purchaseId": "p_001",
        "type": "RSP",
        "date": "2026-06-01",
        "units": 1.0,
        "navAtPurchase": 789.72,
        "note": "定期定額 6月"
      },
      {
        "purchaseId": "p_002",
        "type": "LUMP",
        "date": "2026-05-15",
        "units": 5.5,
        "navAtPurchase": 810.00,
        "note": "單筆買入"
      }
    ]
  }
]
```

**欄位說明：**

| 欄位 | 型別 | 說明 |
|------|------|------|
| `id` | string | 持有記錄唯一 ID（timestamp + random） |
| `fundCode` | string | 基金代碼（如 UNI023） |
| `fundName` | string | 基金中文名稱 |
| `currency` | string | 計價幣別（TWD / USD / ...） |
| `purchases` | array | 所有買入紀錄（定期定額 + 單筆） |
| `purchases[].type` | string | `"RSP"`（定期定額）/ `"LUMP"`（單筆） |
| `purchases[].date` | string | 買入日期 YYYY-MM-DD |
| `purchases[].units` | number | 買入單位數（可手動編輯） |
| `purchases[].navAtPurchase` | number | 買入當日淨值（手動輸入或 API 查詢） |

#### 4.1.2 淨值歷史快取

**Key：** `fund_tracker_nav_cache_<fundCode>`  
**型別：** JSON Object

```json
{
  "fundCode": "UNI023",
  "lastUpdated": "2026-06-10T14:00:00+08:00",
  "navHistory": [
    { "date": "2026-06-10", "nav": 741.47 },
    { "date": "2026-06-09", "nav": 743.21 },
    { "date": "2026-06-06", "nav": 738.55 }
  ]
}
```

**快取策略：**
- 每次開啟 App 時，若距上次更新 > 6 小時，觸發重新抓取
- 每支基金最多保留 **730 筆**（約 2 年）資料
- 節假日無資料為正常現象，不補零

#### 4.1.3 使用者設定

**Key：** `fund_tracker_settings`  
**型別：** JSON Object

```json
{
  "defaultChartRange": "3M",
  "currency": "TWD",
  "refreshOnOpen": true,
  "theme": "light"
}
```

#### 欄位定義

| 欄位名 | 說明 | 對齊 |

| 基金名稱 | 中文名稱（截斷後 tooltip 顯示全名） | 左 |

## 6. 功能模組規格

### 6.1 基金搜尋模組

**輸入：** 使用者輸入的字串（代碼 or 中文名稱）  
**流程：**

1. App 啟動時，從 TDCC API 或政府開放平台載入**全部基金清單**，快取至 `fund_tracker_fund_list`
2. 搜尋時對快取清單進行前端 fuzzy match（代碼完全符合優先，名稱關鍵字次之）
3. 若清單過期（> 7 天）則背景更新

**清單來源 API：**

```
GET https://openapi.tdcc.com.tw/v1/opendata/5-1   （期信基金總覽）
GET https://openapi.tdcc.com.tw/v1/opendata/3-1   （境外基金基本資料）
GET https://data.gov.tw/api/v2/rest/datastore/<dataset_id>  （境內基金）
```

### 6.2 淨值更新模組

**觸發時機：**
- App 開啟時（若距上次 > 6 小時）
- 使用者點擊「⟳ 更新」按鈕
- 手動強制重整（下拉刷新，手機）

**流程：**

```
For each fundCode in holdings:
  1. 呼叫 API 取得最新淨值
  2. 比對 localStorage 快取
  3. 若有新資料 → 追加至 navHistory
  4. 更新 lastUpdated timestamp
  5. 若 API 失敗 → 顯示 "使用快取資料" 提示，不清空舊資料
```

**批次限流：** 若持有基金 > 10 支，每 500ms 發一個請求，避免 API Rate Limit

### 6.3 歷史資料累積策略

由於 TDCC 等 API 僅保留近 1 年資料，需在本地累積：

- 每次更新時，將新取得的淨值**合併**至 localStorage 快取（不覆蓋舊資料）
- 最多保留 730 筆（約 2 年），超過則移除最舊的
- 若使用者清除瀏覽器資料，歷史會遺失（此為 localStorage 方案的已知限制）

### 6.4 資料匯出/匯入

- **匯出：** 將全部 localStorage 資料轉為 JSON 檔案下載
- **匯入：** 上傳 JSON 檔案還原資料（換裝置轉移用）
- 位置：設定頁面

## 7. 計算邏輯

### 7.1 加權平均買入成本

```
持有總單位數 = Σ purchases[i].units
總買入成本   = Σ (purchases[i].units × purchases[i].navAtPurchase)
加權平均淨值 = 總買入成本 ÷ 持有總單位數
```

### 7.2 損益計算

```
現值       = 持有總單位數 × 最新淨值
損益金額   = 現值 − 總買入成本
損益率 (%) = (損益金額 ÷ 總買入成本) × 100
```

### 7.3 時間區間報酬率

```
區間開始淨值 = navHistory 中最接近（≥）開始日期的淨值
區間報酬率  = (最新淨值 − 區間開始淨值) ÷ 區間開始淨值 × 100%
```

### 7.4 當日漲跌

```
當日漲跌(金額) = 今日淨值 − 前一營業日淨值
當日漲跌(%)   = 當日漲跌(金額) ÷ 前一營業日淨值 × 100%
```

若今日淨值尚未更新（e.g. 假日），則使用最近兩筆有資料日期計算。

### 7.5 總損益摘要（Header）

```
總投資成本 = Σ 各基金的 總買入成本（統一換算為 TWD）
總現值     = Σ 各基金的 現值（統一換算為 TWD）
總損益金額 = 總現值 − 總投資成本
總報酬率   = 總損益金額 ÷ 總投資成本 × 100%
```

**幣別換算：** USD、EUR 等外幣計價基金，需乘以對應匯率（可呼叫 TDCC 匯率 API 或 Yahoo Finance 匯率）

## 9. 限制與已知問題

| 項目 | 說明 |
|------|------|
| localStorage 上限 | 約 5MB，持有大量基金且長期累積後可能不足，建議定期匯出 |
| CORS 問題 | TDCC API 可能不允許瀏覽器直接存取，需 Proxy 中轉 |
| 換裝置資料遺失 | localStorage 不跨裝置，需手動匯出/匯入 |
| 境內基金 API 不穩定 | 政府開放平台 API 偶爾維護中斷，需有 fallback 機制 |
| 歷史資料上限 | TDCC 等 API 僅保留近 1 年資料，更早的依賴本地累積 |
| 假日無淨值 | 非營業日不更新，屬正常現象，App 需顯示「最近營業日」提示 |
| 外幣匯率 | 外幣基金換算 TWD 總損益需即時匯率，另行呼叫匯率 API |
